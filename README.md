# 못그린 퀴즈 모음

지금까지 만든 못그린 퀴즈들을 한 번에 소개하고 공유하기 위한 바이럴 허브 페이지입니다.

## 포함된 퀴즈

- 못그린 영화퀴즈
- 못그린 애니퀴즈
- 신비아파트 귀신퀴즈
- 못그린 일본애니퀴즈
- 못그린 한국영화퀴즈

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

GitHub Pages workflow가 포함되어 있습니다. `main` 브랜치에 push하면 자동으로 배포됩니다.

## 이미지 규칙

`public/thumbnails`에는 각 퀴즈에서 직접 만든 패러디 발그림만 복사해 사용합니다.

공식 영화 스틸, 포스터, 로고, 배우 사진, 공식 애니메이션 이미지, 캡처 이미지는 사용하지 않습니다.
